# ⚔️ Agent Arena

Agent Arena is a fast-paced, browser-based auto-battler game where you deploy AI-powered agents and battle against enemies in turn-based combat.

No installation needed — just open the game and play.

---

## 🎮 What is this?

Agent Arena is a strategy game where:

-   You build your field using agent cards
-   Each agent has unique stats and abilities
-   Battles happen automatically when you end your turn
-   Goal: destroy the enemy core before yours is destroyed

---

## 🚀 Features

-   ⚡ Instant play — runs directly in the browser
-   🧠 AI-themed characters with unique abilities
-   🎴 Card-based deployment system
-   ⚔️ Auto-battle combat mechanics
-   🎨 Smooth animations using Framer Motion
-   📜 Real-time combat log

---

## 🕹️ How to Play

1. Open the game (`index.html`)
2. Deploy agents from your hand
3. Click **"Battle!"**
4. Watch combat resolve automatically
5. Repeat until one core reaches 0 HP
